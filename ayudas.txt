OPCIONES COMPARENDO:

FORMULARIO DE LOS COMPARENDOS
https://es.slideshare.net/willl777/manual-de-infracciones-0

-VIA PRINCIPAL Y SECUNDARIA: CONCATENACION ENTRE TIPO DE VIA (AV,CL,CR,AU,DS,TR) Y NUNMERO O NOMBRE
-RADIO ACCION: NACCIONAL, MUNICIPAL
-MODALIDAD DE TRANSPORTE: PASAJEROS COLECTIVO,PASAJEROS  INDIVIDUAL,PASAJEROS  MASIVO,PASAJEROS  ESPECIAL ESCOLAR,PASAJEROS  ESPECIAL ASALARIADO,PASAJEROS  ESPECIAL DE TURISMO,PASAJEROS  ESPECIAL OCASIONAL,MIXTO,CARGA
-TIPO INFRACTOR:CONDUCTOR,PEATON,PASAJERO
-TIPO COMPARENDO: HABILITADO,INFRACCION,ANULADO,CANCELADO,JUICIO,INASISTENCIA

HABILITADO: ES CUANDO EL COMPARENDO NO ESTA CANCELADO Y ESTA ANTES DE LOS 5 DIAS HABILES
INFRACCION: CUANDO SE CUMPLE LOS 5 DIAS HABILES DE LA FECHA DEL COMPARENDO, DESDE AQUI COMIENZA A CORRER EL INTEREZ
ANULADO:ES CAUNDO EL COMPARENDO SE HAYA HECHO POR ERROR ENTONCES NO SE PUEDE ELIMINAR SI NO QUE ANULAR
CANCELADO: CUANDO EL COMPARENDO YA ESTA LIBRE DE CARGOS O SE HAYA CANCELADO
JUICIO: ES CUANDO EL COMPARENDO SE VA JUICIO Y SE DA ESTE CASO CUANDO LA PERSONA NO ACEPTA EL COMPARENDO
INASISTENCIA:ES CUANDO LA PERSONA NO VA AL CURSO

-LAS FORANEAS QUE QUEDAN SE CARGAN DESDE LA BASE DE DATOS

AL MOMENTO DE REGISTRAR
-EL TESTIGO NO ES OBLIGATORIO
-SE COLOCA LA GRUA SI EL TIPO DE INFRACCION ES TAN GRAVE COMO PARA QUE SE REMOLQUE EL VAHICULO



DETALLE COMOARENDO:
ESTA FUNCION ESTA ASIGNADA PARA EL FUNCIONARIO Y SOLO SE LLENA CUANDO LA PERSONA ASISTE A LA A LA ACEPTACION DEL COMPARENDO

PAGO COMPARENDO:
ESTA FUNCION ESTA ASIGNADA PARA EL FUNCIONARIO Y ES CUANDO SE VA A PAGAR EL COMPARENDO ENTONCES SE MIRA EL TIPO DE COMPARENDO, SI ES UNA INFRACCION SE DEBE COBRAR LOS INTERESES MAS EL NUMERO DE SALARIOS MINIMOS QUE INDICA EL TIPO DE INFRACCION Y SI ES HABILITADO ENTONCES SOLO DEBE PAGAR EL NUMERO DE SALARIOS MINIMOS (PARA HACER ESTA TRANSACION SE DEBE RELACIONAR LA TABLA VALORES Y TIPO DE INFRACCION)


LA GRUAS LA REGISTRA EL FUNCIONARIO
NO SE PUEDE ELIMINAR UNA GRUA DONDE ESTA YA HAYA TRABAJADO O ASIGNADO A UN COMPARENDO
CADA VEZ QUE SE LA ASIGNA UN COMPARENDO UNA GRUA SE LE AUMENTA EL CONSECUTIVO 	


LA ASISTENCIOA SE LLENA CUANDO ESTE ACEPTA EL COMPARENDO Y SE LE ASIGNA UN CURSO SI NO VA AL CURSO SE LE COBRARA UN INTERES, EL VALOR DE ESTE INTERES ESTA EN LA TABLA CURSO Y SE LE AÑADE AL PAGO DEL COMPARENDO
